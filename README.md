Phase 1: Requirement and User Story
Project: quizWebApp
1. รายชื่อสมาชิกกลุ่ม

นางสาวปภัสสร ว่องธนากิจ

นางสาวศรุชา วิริยะจิตต์

นางสาวภัทรธิดา จารุจิตจำเริญ

2. ที่มาของปัญหาและความสำคัญ

ในปัจจุบัน การเรียนการสอนออนไลน์มีการใช้ระบบทำข้อสอบออนไลน์อย่างแพร่หลาย
อย่างไรก็ตาม ระบบจำนวนมากยังขาดการวิเคราะห์ผลคะแนนและสถิติที่ชัดเจน เช่น คะแนนเฉลี่ย คะแนนสูงสุด หรือคะแนนต่ำสุด
ส่งผลให้ผู้เรียนไม่สามารถประเมินผลการเรียนรู้ของตนเองได้อย่างมีประสิทธิภาพ

โครงการ quizWebApp จึงมีวัตถุประสงค์เพื่อพัฒนาเว็บแอปพลิเคชันสำหรับทำข้อสอบออนไลน์
ที่สามารถตรวจคำตอบและวิเคราะห์ผลคะแนนได้โดยอัตโนมัติ

3. จุดประสงค์ของโครงการ และประโยชน์ที่คาดว่าจะได้รับ
จุดประสงค์ของโครงการ

เพื่อพัฒนาเว็บแอปพลิเคชันทำข้อสอบออนไลน์ด้วย Node.js

เพื่อสร้างระบบตรวจคำตอบและคำนวณคะแนนอัตโนมัติ

เพื่อแสดงผลสถิติของคะแนนสอบ

ประโยชน์ที่คาดว่าจะได้รับ

ผู้เรียนสามารถประเมินผลการเรียนรู้ของตนเองได้

ลดภาระในการตรวจข้อสอบ

ผู้พัฒนาได้ฝึกการทำงานเป็นทีมและการใช้ GitHub

4. Data Structure ที่คาดว่าจะใช้

Question
ใช้เก็บข้อมูลคำถาม ตัวเลือก และคำตอบที่ถูกต้อง

Quiz
ใช้เก็บชุดข้อสอบ

Result
ใช้เก็บคะแนนและผลลัพธ์ของผู้ใช้

Data structure เหล่านี้จะถูกนำมาใช้ในการตรวจคำตอบและคำนวณคะแนน

5. ขอบเขตของโครงการ (Scope)
สิ่งที่ทำ

ทำข้อสอบแบบปรนัย

ตรวจคำตอบอัตโนมัติ

คำนวณและแสดงคะแนน

แสดงสถิติพื้นฐาน (Average, Max, Min)

สิ่งที่ไม่ทำ

ระบบชำระเงิน

ระบบสมัครสมาชิกขั้นสูง

ระบบสอบที่มีความซับซ้อน

6. Functional & Non-Functional Requirements
Functional Requirements

ผู้ใช้สามารถทำข้อสอบออนไลน์ได้

ระบบสามารถตรวจคำตอบได้

ระบบสามารถคำนวณและแสดงคะแนนได้

Non-Functional Requirements

ระบบใช้งานผ่าน Web Browser

ระบบมีความถูกต้องในการคำนวณ

ระบบตอบสนองได้รวดเร็ว

โค้ดสามารถทดสอบได้
